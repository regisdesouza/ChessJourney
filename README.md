# ChessJourney

Este projeto tem como objetivo introduzir iniciantes ao mundo do xadrez de forma simples e interativa. A aplicação foi desenvolvida utilizando HTML, CSS, JavaScript e Chart.js, combinando ensino e visualização de dados para tornar o aprendizado mais dinâmico.

Além de ensinar as regras básicas e os movimentos das peças, o projeto busca despertar o interesse pelo xadrez e destacar seus benefícios cognitivos, como o desenvolvimento do raciocínio lógico, da concentração e da tomada de decisão.

Criado a partir da minha paixão pelo xadrez e da experiência pessoal de como o jogo me ajudou a crescer mental e emocionalmente, este projeto pretende inspirar outras pessoas a descobrirem o poder transformador do xadrez.
